## ūüéĮ What Is This?

WiFi-Radar is a **real-time, military-style radar interface** that scans every wireless access point near you and displays them as live blips on a rotating radar ‚ÄĒ just like the ones you've seen in war movies, but for your Wi-Fi neighborhood.

Built for **network engineers, security researchers, curious geeks**, and anyone who's ever wondered *"whose router is that?"*

No cloud. No tracking. No nonsense. **100% local.**

## ūü©ļ Troubleshooting

<details>
<summary><b>ūüďĶ APs showing 0 on Windows</b></summary>

1. Make sure Wi-Fi is **ON** in Windows Settings
2. Verify WLAN AutoConfig service is running:
   `Win+R` ‚Üí `services.msc` ‚Üí find **WLAN AutoConfig** ‚Üí Start
3. Test manually in CMD:
   ```
   netsh wlan show networks mode=Bssid
   ```
4. Try running `START_WINDOWS.bat` **as Administrator**
5. Visit `http://localhost:5000/api/debug` to see raw output
</details>

<details>
<summary><b>ūüćé macOS Ventura/Sonoma issues</b></summary>

The `airport` utility was removed in newer macOS versions.
The app automatically falls back to `system_profiler`.

If still failing:
- Go to **System Preferences ‚Üí Privacy & Security ‚Üí Full Disk Access**
- Add your Terminal app
</details>

<details>
<summary><b>ūüźß Linux ‚ÄĒ permission denied</b></summary>

```bash
sudo python3 server.py
# or
sudo nmcli dev wifi list --rescan yes
```

Make sure NetworkManager is running:
```bash
sudo systemctl start NetworkManager
```
</details>

<details>
<summary><b>ūüĒć Why does my router show multiple blips?</b></summary>

Modern **WiFi 6/6E/7 routers** with MLO (Multi-Link Operation) broadcast the **same SSID from multiple physical radios** (2.4 GHz, 5 GHz, 6 GHz) ‚ÄĒ each with its own MAC address.

WiFi-Radar detects this and **groups them** under one entry in the list, showing each band as a sub-row. On the radar, one label is shown at the strongest radio with a `3√ó 2.4+5+6G` badge.
</details>
